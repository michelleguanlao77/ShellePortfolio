# Michelle D. Guanlao - Portfolio Website

A modern, responsive portfolio/resume website for Michelle D. Guanlao.

## Features

- **Unique Design**: Custom gradient backgrounds and modern card-based layout
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive**: Smooth animations, hover effects, and interactive elements
- **Print-Friendly**: Optimized for printing as a resume
- **Fast Loading**: Lightweight and optimized for performance

## Sections

1. **Header**: Profile information with contact details
2. **Personal Information**: Complete personal details in an organized grid
3. **Educational Background**: Timeline view of educational journey
4. **Skills & Competencies**: Technical and soft skills categorized

## Technologies Used

- HTML5
- CSS3 (with modern features like Grid, Flexbox, and CSS animations)
- JavaScript (ES6+)
- Font Awesome icons
- Google Fonts (Poppins)

## Setup Instructions

1. Place all files in your web server directory (e.g., `htdocs` for XAMPP)
2. Open `index.html` in a web browser
3. Or access via `http://localhost/Michelle/` if using a local server

## File Structure

```
Michelle/
├── index.html          # Main HTML file
├── style.css          # CSS styling
├── script.js          # JavaScript functionality
└── README.md          # This file
```

## Customization

- Update contact information in the HTML file
- Modify colors by changing CSS custom properties
- Add or remove sections as needed
- Replace placeholder profile image with actual photo

## Browser Compatibility

- Chrome (recommended)
- Firefox
- Safari
- Edge
- Mobile browsers
